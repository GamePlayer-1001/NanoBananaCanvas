## 一、项目概述

### 1.1 项目名称

Nano Banana Canvas

### 1.2 项目定位

可视化 AI 工作流编排平台，用户通过拖拽节点、连接管道组合各类 AI 能力，构建自动化工作流。对标 [martini.art](https://martini.art/)（YC W26），提供积分制 AI 执行 + 社区分享功能。

### 1.3 目标用户

- AI 爱好者、内容创作者
- 需要批量处理 AI 任务的用户
- 希望低代码使用 AI 的非技术用户
- 希望学习和复用他人工作流的用户
- 国际用户（中英双语支持）

### 1.4 参考产品

| 产品        | 特点                       | 我们的差异化                     |
| ----------- | -------------------------- | -------------------------------- |
| martini.art | 节点式 AI 视频创作，积分制 | 直接对标，更注重国内 AI 模型生态 |
| ComfyUI     | 开源本地工作流             | 我们是云端 SaaS，无需本地 GPU    |
| Dify        | AI 应用构建                | 我们更偏创意工作流而非应用构建   |
| n8n         | 自动化工作流               | 我们聚焦 AI 多模态而非通用自动化 |

### 4.3 双模式 AI 执行

**问题**：需要同时支持"平台积分"和"用户自带 Key"两种模式

**解决方案**：

```typescript
// Worker AI 代理路由
app.post('/api/ai/execute', authMiddleware, async (c) => {
  const { model, input, mode, userApiKey } = await c.req.json()
  const userId = c.get('userId')

  if (mode === 'credits') {
    // 模式一：积分制
    const pricing = await getModelPricing(c.env.DB, model)
    const balance = await getCreditBalance(c.env.DB, userId)

    if (balance < pricing.credits_per_call) {
      return c.json({ error: 'Insufficient credits' }, 402)
    }

    // 冻结积分
    await freezeCredits(c.env.DB, userId, pricing.credits_per_call)

    try {
      const result = await callAI(model, input, c.env[`${model.provider}_API_KEY`])
      // 扣费
      await deductCredits(c.env.DB, userId, pricing.credits_per_call)
      return c.json(result)
    } catch (error) {
      // 失败退还
      await unfreezeCredits(c.env.DB, userId, pricing.credits_per_call)
      throw error
    }
  } else {
    // 模式二：用户自带 Key
    const result = await callAI(model, input, userApiKey)
    return c.json(result)
  }
})
```

**可行性**：✅ 核心是 API Key 路由逻辑，无技术瓶颈

### 4.4 Stripe 订阅集成

**问题**：需要月度订阅 + 积分包一次性购买

**解决方案**：

```typescript
// 创建 Checkout Session
import Stripe from 'stripe'

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!)

// 订阅
export async function createSubscriptionCheckout(userId: string, plan: string) {
  const session = await stripe.checkout.sessions.create({
    mode: 'subscription',
    line_items: [{ price: PLAN_PRICE_IDS[plan], quantity: 1 }],
    success_url: `${APP_URL}/billing?success=true`,
    cancel_url: `${APP_URL}/billing?canceled=true`,
    metadata: { userId, plan },
  })
  return session.url
}

// Webhook 处理
export async function handleStripeWebhook(event: Stripe.Event) {
  switch (event.type) {
    case 'invoice.paid':
      // 订阅续费成功 → 充值月度积分
      await addMonthlyCredits(event.data.object)
      break
    case 'customer.subscription.deleted':
      // 订阅取消 → 降级为 Free
      await downgradeToFree(event.data.object)
      break
  }
}
```

**可行性**：✅ Stripe 标准集成模式

### 4.6 Cloudflare Queues（异步任务）

**问题**：视频生成等任务需要 2-10 分钟

**解决方案**：

```typescript
// 提交任务
app.post('/api/tasks', authMiddleware, async (c) => {
  const task = createTask(...)
  await c.env.DB.prepare('INSERT INTO async_tasks ...').run()
  await c.env.AI_TASK_QUEUE.send({ taskId: task.id })
  return c.json({ taskId: task.id })
})

// Worker Queue 消费者
export default {
  async queue(batch, env) {
    for (const msg of batch.messages) {
      const { taskId } = msg.body
      await processTask(taskId, env)
      msg.ack()
    }
  }
}
```

**可行性**：✅ Cloudflare Queues 原生支持
