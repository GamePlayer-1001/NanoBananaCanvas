```
┌─────────────────────────────────────────────────────┐
│  Logo    │   🔍 搜索广场作品...          │  👤 头像 │  ← 顶部栏
├──────────┼──────────────────────────────────────────┤

│  广场    │                                          │

│  个人    │                                          │
│    ↑     │                                          │
│  侧边栏  │                                          │
└──────────┴──────────────────────────────────────────┘
```

### 8.2 页面路由

| 页面 | 路由 | 访问权限 | 说明 |
|-----|------|---------|------|
| 首页 | `/` | 公开 | 产品介绍 |
| 广场 | `/explore` | 公开 | 用户分享的成果列表（瀑布流） |
| 作品详情 | `/explore/:id` | 公开 | 作品介绍/分享者/节点类型/点赞/收藏/运行/举报 |
| 创作空间 | `/workspace` | 需登录 | 作品管理（编辑/分享/删除）+ 新增作品 |
| 画板编辑器 | `/workspace/:id` | 需登录 | 画板（导入/导出/设置/API配置） |
| 个人 | `/profile` | 需登录 | 已创建作品/已分享作品/已收藏作品/账号管理 |

### 8.3 广场页面结构

```
广场
├── 列表页 /explore

│   └── 排序

└── 详情页 /explore/:id
    ├── 作品介绍（标题、描述）

    └── 操作按钮

        ├── 收藏

        └── 举报
```
