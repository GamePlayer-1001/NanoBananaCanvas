## 一、阶段划分

| 阶段 | 名称         | 目标                                 | 状态                     |
| ---- | ------------ | ------------------------------------ | ------------------------ |
| MVP  | 最小可行产品 | 验证核心工作流逻辑                   | ✅ M1~M5 全部完成 |

## 当前现状校准（2026-04-21）

1. 当前活跃主链是：`Clerk/访客双身份 → users 表 actor 收口 → account 账户页 → workspace/folders → async_tasks + R2 输出归档`。
2. `ProfileModal`、`Billing/Stripe/Credits` 属于历史阶段遗留或文案残留，不再代表当前运行时主链；本清单继续保留其历史记录，但会明确标注是否已归档。
3. 当前代码确认存在并被主链消费的关键增量包括：`/account` 账户页、文件夹分组、`/api/folders`、`/api/workflows/:id/history`、`/api/workflows/:id/thumbnail`、`/api/files/[...key]`、Clerk Webhook 与账户级任务输出回收。
